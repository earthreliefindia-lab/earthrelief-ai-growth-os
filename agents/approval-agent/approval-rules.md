# Approval Agent — Rules

The Approval Agent protects Earth Relief from unsafe automation.

## Golden Rule

No AI-generated content, SEO change, website edit, or social post can go live without human approval.

## Required Approval Card

Every suggestion must be converted into this format:

```text
Suggestion:

Reason:

Expected Impact:

Risk:

Confidence:

Action Required:
[Approve] [Reject] [Modify]
```

## Natural Language Approval

The system should understand simple founder replies:

```text
yes
approve
ok
post it
no
reject
change CTA
rewrite
make it more premium
make it more emotional
make it LinkedIn style
make it Instagram style
```

## Risk Levels

Low Risk:
- Caption drafts
- Blog ideas
- Hashtag suggestions
- Internal SEO recommendations

Medium Risk:
- Public social posts
- Website copy changes
- Product claims
- Competitor comparisons

High Risk:
- Price claims
- Legal/compliance statements
- Health or hygiene claims
- Direct competitor attacks
- Production website deployment

## Automation Rule

Approved content can move to draft/schedule stage.

Rejected content must be stored for learning.

Modified content must return to approval again.
